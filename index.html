之前一直有做个博客的想法，但是因为种种原因(主要是懒)，一直没有弄，今天趁周末没事情做，搭个博客记录一写工作中遇到的问题，和一些知识点，最主要的是督促自己进步，以后尽量一周一更，先立个flag。

创建时遇到的问题

探索未知的事物总不是一帆风顺的，我也一样，当时创建时也遇到很多稀奇古怪的问题，好在最后都解决了，如果你跟我一样也遇到这样的问题，希望能帮助到你

修改主题

创建好博客之后，修改主题的时候老是不生效，一直都是默认主题，如果你也想修改一款Hexo上好看的主题的话，教程如下：

  • 首先在Hexo上选择一个你想要的主题
  • 拿我这个主题hexo-theme-particle举例
    // 先把选好的主题clone到你的博客source文件夹下,默认主题可以不要
$ git clone https://github.com/korilin/hexo-theme-particle.git particle
  • 修改显示主题
    在项目目录下查找 _config.yml 找到theme，修改如下：(theme:主题名字)
    # Extensions
$ Plugins: https://hexo.io/plugins/
$ Themes: https://hexo.io/themes/
$ theme: hexo-theme-particle
  • 然后重启项目
    # 清除缓存文件
$ hexo cl
# 生成静态文件
$ hexo g
# 启动服务
$ hexo s

详细内容请参考Hexo文档

修改主题内容

现在你的主题应该可以使用了，但是还有很多细节需要设置，比如网站名称，标题、副标题，这个要怎么设置呢？
你可以在项目目录下的 _config.yml 中修改大部分的配置。

# Site
title: 网站标题
subtitle: 网站副标题
description: 网站描述
keywords: 网站的关键词。可以多个关键词。
author: 您的名字
language: 网站使用的语言。常见的有 zh-Hans和 zh-CN。
timezone: 网站时区。Hexo 默认使用您电脑的时区。

其他的不常用就不一一列举了，详细内容请参考Hexo文档
